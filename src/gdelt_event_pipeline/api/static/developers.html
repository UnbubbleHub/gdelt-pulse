GDELT Pulse API

Hybrid semantic + keyword search over global news events. All endpoints are read-only (GET).

Quick Start

X-API-Key: <your-key>
30 req / 60 s per IP

The X-API-Key header is only required when the server has an API_KEY environment variable set. Local dev instances and public deployments without a key set allow unauthenticated requests.

curl "/api/stats" \
  -H "X-API-Key: your-key-here"

Endpoints

Core

MethodPathKey paramsDescription
GET /api/stats — Total articles, clusters, embeddings, largest cluster
GET /api/articles limit (1–200, default 50) Recent articles, newest first
GET /api/clusters limit, sort, location, person, org, theme, domain, source, date_from, date_to Active event clusters with optional filtering
GET /api/clusters/{id} — Single cluster with all member articles
GET /api/search q (required), limit, semantic_weight, clusters, location, person, org, theme, domain, source, date_from, date_to Hybrid semantic + keyword search via RRF

Globe

MethodPathKey paramsDescription
GET /api/globe/clusters mode (live|rising|silent), limit Top clusters with geographic coordinates for 3D globe rendering

Polarization

MethodPathKey paramsDescription
GET /api/polarization limit, min_articles Story clusters ranked by narrative tone divergence across sources
GET /api/polarization/{id} — Per-article tone breakdown for a single cluster, grouped by source

Attention Asymmetry

MethodPathKey paramsDescription
GET /api/asymmetry — Coverage volume vs crisis intensity by country; reveals over- and underreported regions

Geopolitical Gravity

MethodPathKey paramsDescription
GET /api/gravity/graph min_weight, limit_edges Country co-mention graph (nodes + edges) for gravity map
GET /api/gravity/country/{code} — Single country: top connections, top clusters, avg tone (ISO 3166-1 alpha-2 code)

Source DNA

MethodPathKey paramsDescription
GET /api/sources/fingerprints limit, sort, min_articles Per-source tone fingerprints, top themes, and top countries
GET /api/sources/{domain}/detail — Full fingerprint for one source domain: tone timeline, all themes, category breakdown

Story Propagation

MethodPathKey paramsDescription
GET /api/propagation/stories limit, min_sources Multi-source clusters suitable for propagation analysis
GET /api/propagation/{id} — Chronological timeline of how a story spread across sources

Topic Velocity

MethodPathKey paramsDescription
GET /api/velocity/topics hours (6–168, default 48), limit Themes with rising or declining velocity over a configurable time window
GET /api/velocity/timeline theme (required), hours Hourly article count for a specific GDELT theme code

Examples

Search

curl "/api/search?q=earthquake+Turkey&semantic_weight=0.7&location=Turkey&limit=10" \
  -H "X-API-Key: your-key-here"

List clusters with filters

curl "/api/clusters?sort=articles&theme=MILITARY_CONFLICT&limit=20" \
  -H "X-API-Key: your-key-here"

Globe data (rising stories)

curl "/api/globe/clusters?mode=rising&limit=12" \
  -H "X-API-Key: your-key-here"

Polarization ranking

curl "/api/polarization?limit=20&min_articles=30" \
  -H "X-API-Key: your-key-here"

Attention asymmetry

curl "/api/asymmetry" \
  -H "X-API-Key: your-key-here"

Country gravity detail

curl "/api/gravity/country/US" \
  -H "X-API-Key: your-key-here"

Source fingerprint

curl "/api/sources/bbc.co.uk/detail" \
  -H "X-API-Key: your-key-here"

Story propagation timeline

curl "/api/propagation/stories?min_sources=5&limit=10" \
  -H "X-API-Key: your-key-here"

Topic velocity (rising themes)

curl "/api/velocity/topics?hours=24&limit=20" \
  -H "X-API-Key: your-key-here"

For full parameter details and to try requests in the browser, use the interactive Swagger UI.
